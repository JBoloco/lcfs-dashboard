## LCFS Dashboard

* This project utilised streamlit version 0.8.11 to create a simple dashboard

## General Information
* It uses cross-sectional data from the LCFS database to produce charts of weekly income and expenditure
* There are filters in the different categories to change the output of the charts

## Setup
* To run the project, you will need to ensure you have the relevant packages installed. You can do this using your command-line/terminal. An example for installing Streamlit is included below:

```
$ pip install streamlit
```

* To actually run the project, you will need to ensure you are in the saved project's directory in your command-line/terminal and then run:
```
$ streamlit run dashboard.py
```
